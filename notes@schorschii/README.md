# Note Desklet

## Basic markdown support

1. Headlines

    Write headlines like this:
    ```
    # Headline 1
    ## Headline 2
    ### Headline 3
    #### Headline 4
    ##### Headline 5
    ###### Headline 6
    ```
2. Bold text

    Write **bold text** like this:
    ```
    **text**
    ```

3. Italic text

    Write *italic text* like this:
    ```
    *text*
    ```
4. Strikethrough text

    Write ~~strikethrough~~ text like this:
    ```
    ~~text~~
    ```
5. Monospace text

    Write `monospace` text like this:
    ```
    `text`
    ```
