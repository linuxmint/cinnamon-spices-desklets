# Stock Price Chart Desklet

## Stock Price Chart Desklet shows price of a single ticker symbol as a chart on your desktop.

### It fetches data from Yahoo Finance and displays it in a simple line chart. No need for an API key or account. The request simulates a browser request to avoid restrictions.

## Features:
* Stock price for a period of 1d intervals for the configurable number of days from 1 to 31.
* Configurable value line color.
* Configurable desklet scale.
* Configurable font size and color.
* Configurable background color and transparency.
* Update interval is fixed to 1h since the period values are 1d each, there is no need for more frequent updates. If you need to update the data there is a button to do it manually.
