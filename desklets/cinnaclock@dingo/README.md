# Cinnaclock Desklet

An elegant, modern, and highly customizable clock desklet for the Cinnamon desktop environment. **Cinnaclock** allows you to display time, day, and date on separate layers, giving you total control over typography and positioning.

![Cinnaclock Screenshot](https://raw.githubusercontent.com/linuxmint/cinnamon-spices-desklets/master/desklets/cinnaclock@dingo/screenshot.png)

## üåü Features

* **Layered Design:** Time, Day, and Date are separate elements that can be toggled on or off independently.
* **Typography Control:** Choose any system font, size, and color for each element.
* **Advanced Positioning:** Fine-tune the vertical offset of each layer to create unique overlapping effects.
* **Text Effects:** Customizable text shadows (horizontal/vertical offset, blur, and color).
* **Format Flexibility:** Supports standard strftime formats (e.g., `%H:%M`, `%A`, `%e %B %Y`).
* **Automatic Uppercase:** Optional toggle to force uppercase on any text element.
* **Multilingual:** Fully translated into multiple languages.

## üõ† Installation

### Via Cinnamon Settings (Recommended)
1. Right-click on your desktop and select **Add Desklets**.
2. Click on the **Download** tab.
3. Search for **Cinnaclock**.
4. Click the download icon, then add it to your desktop from the **Manage** tab.

### Manual Installation
1. Download the repository.
2. Copy the `cinnaclock@dingo` folder to `~/.local/share/cinnamon/desklets/`.
3. Enable it from the Cinnamon Desklets settings.

## ‚öôÔ∏è Configuration

Once added, right-click the desklet and select **Configure**. You will find four tabs:
* **General:** Toggle visibility, set date/time formats, and adjust vertical positioning.
* **Time:** Change font, color, uppercase mode, and shadow for the clock.
* **Day:** Style the weekday name layer.
* **Date:** Style the calendar date layer.
